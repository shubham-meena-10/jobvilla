# to run the database file in your system follow the given steps 
    1. Install JSON Server.
        Open your Terminal. Execute the following command.  " npm install -g json-server ".
    
    2. Run This command on your react project folder diractry 
        " json-server --watch db.json --port 4000 "
        
    3. To get more information about json-server visit this website
        " https://www.npmjs.com/package/json-server "
    
always keep running this server when you use the app
